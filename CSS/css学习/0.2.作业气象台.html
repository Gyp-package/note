中央气象台继续发布暴雨蓝色预警 江西福建等局地有大暴雨

2022-05-23 17:50:22 来源：中国气象台
原创未经授权不得转载

中国天气网讯 中央气象台5月23日18时继续发布暴雨蓝色预警：

预计， 5月23日20时 至24日20时，湖南南部、江西南部、福建中西部、云南东南部、广西北部和西部、广东北部、 海南岛西北部以及台湾岛西北部等地的部分地区有大到暴雨，其中，江西东南部、福建西部等地局地有大暴雨(100~110 毫米)。 上述部分地区伴有短时强降水(最大小时降雨量20~40毫米，局地可超过50毫米)，局地有雷暴大风等强对流天气。

防御指南:
1、政府及相关部门按照职责做好防暴雨准备工作:
2、学校、幼儿园采取适当措施，保证学生和幼儿安全:
3、驾驶人员应当注意道路积水和交通阻塞，确保安全;
4、检查城市、农田、鱼塘排水系统，做好排涝准备。
