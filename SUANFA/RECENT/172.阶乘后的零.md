给定一个整数 n ，返回 n! 结果中尾随零的数量。

提示 n! = n _ (n - 1) _ (n - 2) _ ... _ 3 _ 2 _ 1

示例 1：

输入：n = 3
输出：0
解释：3! = 6 ，不含尾随 0
示例 2：

输入：n = 5
输出：1
解释：5! = 120 ，有一个尾随 0

本身有 0 的数恰好是 5 的倍数，拥有的 0 的个数恰好又与是 5^n 的倍数的 n 相等,只需要计算阶乘中各个 5 的次方数的个数和

```js
var trailingZeroes = function (n) {
  let count = 0;
  while (n > 0) {
    n = n / 5;
    count += n;
  }
  return count;
};
```
