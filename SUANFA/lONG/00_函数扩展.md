参数变量默认声明，不能再次let,coust声明 
与解构函数默认值结合：函数的对象参数外等于一个空对象
function foo({x, y = 5} = {}) {
  console.log(x, y);
}

foo() // undefined 5


函数的length属性：返回参数个数，
若指定默认值那么包括该位及后续的length均失真
length属性不包含rest参数


局部作用域参数与全局作用域变量，参数与变量指向的作用域不同
箭头函数作为函数指向全局作用域，形成局部死区，值为外部值


箭头函数
var f = v => v;
函数名 参数  返回值
// 等同于
var f = function (v) {
  return v;
};
